# PDF Processing Skill

When the user asks to process a PDF file:

1. Locate the PDF file in the current directory or subdirectories
2. Use appropriate tools to read/extract PDF content
3. Summarize the key points in a structured format
4. If the PDF contains code, provide the code snippets

## Available Tools
- Bash (for running pdftotext or similar tools)
- Read (for reading extracted text files)
- Glob (for finding PDF files)
