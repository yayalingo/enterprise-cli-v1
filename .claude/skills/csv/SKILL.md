# CSV Processing Skill

When the user asks to process CSV data:

1. Use Glob to find CSV files in the project
2. Use Read to inspect CSV content
3. Use Bash to run Python pandas or similar tools for analysis
4. Summarize findings in a structured format

## Available Tools
- Glob (for finding CSV files)
- Read (for reading CSV content)
- Bash (for running analysis scripts)
